# device-supplier-tracker

A tool that allows you to track a stock of devices that I order from a supplier. The tool should allow you to specify a supplier, number of items and type of device.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/device-supplier-tracker.git
cd device-supplier-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
